# Memory Convention

Write down what should survive the session.

## Record

- decisions that change future behavior
- environment facts that will matter again
- mistakes that should not be repeated
- verification results that establish a baseline

## Do Not Record

- raw command spam
- private secrets unless explicitly requested
- temporary guesses
- duplicated notes already captured elsewhere

## Preferred Targets

- `workspace/memory/YYYY-MM-DD.md` for session facts
- `workspace/MEMORY.md` for curated long-term context when applicable
- the relevant skill or config file when the lesson is procedural
